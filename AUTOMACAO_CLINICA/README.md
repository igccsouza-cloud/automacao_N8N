SISTEMA DE AUTOMAÇÃO PARA CLÍNICA ODONTOLÓGICA

Automação completa de gestão de clínica odontológica desenvolvida com n8n, integrando WhatsApp, Google Calendar

# Funcionalidades:

- Cadastro automático de pacientes via WhatsApp;
- Agendamento, cancelamento e reagendamento de consultas;
- Sincronização automática com Google Calendar;
- Lembrete mensal de manutenção;
- Lembrete de não comparecimento à consulta;
- Expectativa de pagamento atualizada;

# Stack:

- N8N (orquestração);
- Evolution API (WhatsApp);
- Google Sheets (banco de dados);
- Google Calendar (agendamento);
- OpenAI GPT-4o mini (assistente);

# Estrutura:

- Fluxo de principal (atendimento e banco de dados);
- Fluxo de agendamentos;
- Fluxo de lembrete mensal;
- Fluxo de lembrete de não comparecimento;

Todos inclusos e um único arquivo, sendo ativados por seus respectivos triggers.

# Como usar:

- Importe o arquivo .json do workflow;
- Configure as credenciais;
- Ative o workflow;
